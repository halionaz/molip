# molip-frontend

## UI 계획

- 좌측 사이드 바
    - Main Bar (메인 바에서 선택한 기능이 Sub Bar에 표시됨)
        - mol.ip 프로덕트 로고 (비활성, 버튼 아님)
        - Document Tree
            - switch to Node Graph (Beta)
            - 즐겨찾기
            - 전체 페이지 트리
        - 전체 문서 검색 기능
        - 전체 업데이트 히스토리 열람 기능 (Git Style)
            - 깃허브 커밋 히스토리처럼 볼 수 있음
    - Sub Bar
- 메인 페이지
    - 현재 페이지 이름 & 저장 상태 표시
    - 에디터
- 우측 사이드 바
    - 프로필 설정
    - 환경 설정
    - 페이지 목차 or 페이지 내 검색 or 페이지 히스토리
    - 도움말, 고객지원 (우측 하단 버튼)
